# 💬 Market Sentiment Analyser

> An NLP pipeline that classifies product reviews as **positive**, **negative**, or **neutral**
> using HuggingFace Transformers.

---

## 🎯 What It Does

Processes raw product review text and outputs:
- Sentiment label (positive / negative / neutral)
- Confidence score
- Batch processing support for large review datasets

## 🛠️ Tech Stack

- **Language:** Python 3.10+
- **NLP:** HuggingFace Transformers
- **Data:** pandas
- **Visualisation:** matplotlib, wordcloud
